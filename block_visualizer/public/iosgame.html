iOS Collection Game

A native iOS game developed with Swift and UIKit, implementing the MVC architecture pattern and SQLite for efficient data management and persistent storage.

🎯 Key Features

  • 🎮 Interactive game world with touch controls
  • 🗡️ Weapon collection system
  • 💾 Database-driven inventory management
  • 📊 Custom progress tracking
  • 💬 Interactive popup notifications
  • 🔍 Weapon filtering system

🎮 Game Features

Core Gameplay

  • Explore the game world using simple touch controls
  • Collect unique weapons with different stats
  • Interact with various objects:
    • 📦 Storage Box: View and manage your weapon collection
    • 🏺 Pot: Discover and collect new weapons
    • 💻 Computer: Search weapons by damage threshold

Movement Controls

→ Tap right side to move right
← Tap left side to move left
↑ Tap top to move up
↓ Tap bottom to move down

📊 Weapon System

Weapon Properties

  • 🏷️ Weapon Name
  • ⚔️ Damage Value
  • ✨ Rarity Rating
  • 🔑 Unique ID

🛠 Technical Features

  • 📱 Built using Model-View-Controller (MVC) pattern
  • 🎨 Custom view implementation for game world rendering
  • 💾 SQLite database integration
  • 🖼️ Interactive UI elements including progress bars and popups

Built with Swift and UIKit
