Disk Scheduling Visualization

This project visualizes key disk scheduling algorithms used in operating systems to manage disk I/O requests. The algorithms are implemented in C for performance and low-level control. Efficient scheduling can be seen with fewer jumps in position, showing improved performance. Check out the source code for more details. FCFS (First Come, First Served), SSTF (Shortest Seek Time First), LOOK, and C-LOOK (Circular LOOK).
